# Tax Calculator

This is a web-based tax calculator application that calculates income tax based on user input for gross annual income, extra income, deductions, and age.

## Features

- Calculates income tax based on specified criteria.
- Validates input fields and displays error messages.
- Displays tax calculation result in a modal.

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/MayurCode2/tax-calculator
   ```

2. Navigate to the project directory:

   ```bash
   cd tax-calculator
   ```

### Usage

1. Open the `index.html` file in a web browser.
2. Fill in the required fields:
   - Gross Annual Income: Enter your gross annual income before deductions.
   - Extra Income: Enter any additional income apart from your gross income.
   - Deductions: Enter any deductions or exemptions you are eligible for.
   - Age: Select your age range (<40, ≥40 & <60, ≥60).
3. Click the "Calculate Tax" button to see the tax calculation result in a modal.
